A comprehensive, hands-on workshop that transforms how you collaborate with AI as a creative teammate. Through a practical SDG-focused web application project, participants master advanced prompt engineering techniques and learn to move beyond simple commands to start meaningful conversations with AI that amplify problem-solving and creative abilities.

## üìã The 9-Component Prompt Engineering Framework

| Component | Purpose | Applied In | Impact |
|-----------|---------|------------|---------|
| **1. ROLE** | Define AI expertise and perspective | All chats | Sets knowledge level and approach |
| **2. CONTEXT** | Provide background and constraints | All chats | Ensures relevant, targeted responses |
| **3. TASK** | Specify clear objectives and deliverables | All chats | Creates measurable outcomes |
| **4. EXAMPLES** | Show desired formats and patterns | Chat 03, 04 | Improves consistency and quality |
| **5. OUTPUT FORMAT** | Define structure and presentation | Chat 03, 04 | Ensures usable, professional results |
| **6. TONE** | Set communication style and voice | Chat 03, 04 | Matches audience and purpose |
| **7. MODELS** | Select optimal AI capabilities | All chats | Maximizes performance for task type |
| **8. FILES** | Reference relevant documents/media | Chat 02, 04 | Provides additional context and constraints |
| **9. TOOLS** | Leverage available AI features | Chat 02, 04 | Enables enhanced functionality |
